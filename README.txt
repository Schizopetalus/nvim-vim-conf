**************  Configuration commune vim / nvim  ****************

Config nvim dans la mesure du possible adaptable à vim (configs spécifiques à neovim dans un fichier
à part).

Gestion des plugins par vim-plug : avantage la config ne contient que la référence du plugin le plugin
lui-même est installé par un :PlugInstall dans vim ou nvim

Plusieurs confs possibles de plugins : choisir dans le init.vim ou le .vimrrc en sourçant les bons fichiers
